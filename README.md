# HR Analytics Dashboard – Attrition Analysis
## HR Analytics Dashboard using Power BI

## 📊 Project Overview
This project analyzes employee attrition using Power BI to identify key factors affecting employee turnover.

## 🔍 Key Insights
- Total Employees: 1480
- Attrition Rate: 16.1%
- High attrition in specific job roles
- Age group and salary-based attrition trends

## 🛠 Tools Used
- Power BI
- Excel

## 📈 Features
- Attrition by Job Role
- Attrition by Age Group
- Salary Analysis
- Education Field Insights
